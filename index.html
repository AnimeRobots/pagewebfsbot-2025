About this bot & features

A Telegram bot designed to store posts and documents, accessible through special links. This bot provides a convenient way to manage and share content within Telegram.

Fast

Low-latency responses and memory-friendly runtime.

Ads-Token

Customizable Ads token system supports various sites.

Fully Customizable

Pluggable commands, easy variables setup while bot is running.

Multi-ForceSub

Ability to add/remove 20+ groups/channels (Join/Request) as forcesub.

Auto-Delete

Auto delete files to protect from copyright after configurable time.

Protect Content

Prevent users from forwarding file from bot.

Custom Caption

Add custom captions to each file, with file name or original caption.

Statistics

Detailed bot stats of database and server.

Virtual Admins

Add users as admin who can generate links

Broadcast

Send message to all users with full report of broadcast.

/broadcast
Devs Owner

Broadcast a message to all users who have interacted with the bot. This command allows you to send important updates or announcements to all users. You must reply to a message with cmd.

  • $forward : forward the message to all users else copy message (as bot)
  • $pin : pin the message in the channel where bot is running.
  • $keepids : Do not delete users from databse.
  • $testmode : Sends message and quickly deletes it ($pin is ignored).

Tip: Use /formatb to format the message before broadcasting.

Arguments: ($pin / $forward / $keepids / $testmode) 
/broadcast Arguments(Str, CaseSensitive) (ReplyToMessege)
